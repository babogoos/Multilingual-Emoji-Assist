# **App Name**: Emoji Assist

## Core Features:

- Emoji Suggestion: Utilize the Gemma 3B model to suggest relevant emojis based on user inputted text.
- Emoji Display: Display suggested emojis in a scrollable format, grouped by relevance.
- Emoji Copy: Allow users to copy the selected emoji to the clipboard.
- Offline Capability: Provide offline functionality by running the Gemma 3B model locally on the device.

## Style Guidelines:

- Primary color: Indigo (#4B0082) to suggest intelligence and insight, contrasting well with the light background.
- Background color: Light gray (#F0F0F0) for a clean and distraction-free interface.
- Accent color: Yellow-green (#9ACD32), as this vibrant color will draw the eye while complementing the app's mission of connection and translation.
- Body and headline font: 'PT Sans' (sans-serif) for clear readability in both headers and text.
